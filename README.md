# react-native-tweet-embed

Embed a specific tweet in a React Native view

## Installation

```sh
npm install react-native-tweet-embed
```

## Usage

```js
import { multiply } from "react-native-tweet-embed";

// ...

const result = await multiply(3, 7);
```
